黑盾杯学习1（登录框问题）

一、遇到登录框(先查看源码)
1.考虑万能密码——用户名任意，密码为’or’1’=’1
2.出现本地管理员，则应想到本地网口127.0.0.1，构造一个数据包（http请求包x-forwared-for：cilent-ip，
proxy-ip——记录数据包经过的代理服务器的地址）ps：管理员系统用户名可能为admin
！！！源代码需注意右边有无滑滑，防止其放置多个空行
若出现base64 md5 url等编码 若有字母大小写，数字，结尾两个==则为base64后寻找在线解码器且解码结果则为账号/密码
而后打开火狐开始burpsuit，打开proxy进行拦截抓包后发送给重发器并在user前输入X-Forwarded-For:127.0.0.1（与user之间必须空一行）
如弹出再试一次，则将上面的密码输入再次发送
！！暴力破解
打开Burp，抓包后右键send to intruder。设置目标地址和端口后，打开positions，Attack type选择Sniper，
然后选择password，在username处直接输入admin在payload里面设置Payload选Simple list，将密码字典导入，start attack。
完成后，按Length排列，发现密码是123456时长度和其他不同，所以直接查看response，发现答案。

指针，结构与函数

指针与函数的变量交换
void swap(int px,intpy)
{
int t;
t=*px;
*px=py;
py=t}
ps:通过指针实现了变量从函数中的输出，倘若不使用指针，则函数中交换的只是主函数中变量的克隆体，
无法返回至主函数中。即必须交换px与py的值

结构与函数的变量改变
1.struct point{
int x;
int y;
}struct point getstruct(void);
int main{
struct point y={0,0};
y=getstruct(); } \此处使主函数中的y直接＝函数中返回的结构变量来达到改变主函数变量的目的
struct point getstruct(void)
{
struct point p;
scanf(“%d %d”,&p.x,&p.y);
return p;
}
2..struct point{
int x;
int y;
}struct point getstruct(struct point);
int main{
struct point y={0,0};
getstruct(&y);
}
struct point getstruct(struct point)
scanf(“%d %d”,&p->x,&p->y);
printf(“%d %d”,p->x,p->y);
return p;\使用结构指针来使得结构变量变化
