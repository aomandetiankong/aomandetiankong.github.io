指针，结构与函数

指针与函数的变量交换
void swap(int px,intpy)
{
int t;
t=*px;
*px=py;
py=t}
ps:通过指针实现了变量从函数中的输出，倘若不使用指针，则函数中交换的只是主函数中变量的克隆体，
无法返回至主函数中。即必须交换px与py的值

结构与函数的变量改变
1.struct point{
int x;
int y;
}struct point getstruct(void);
int main{
struct point y={0,0};
y=getstruct(); } \此处使主函数中的y直接＝函数中返回的结构变量来达到改变主函数变量的目的
struct point getstruct(void)
{
struct point p;
scanf(“%d %d”,&p.x,&p.y);
return p;
}
2..struct point{
int x;
int y;
}struct point getstruct(struct point);
int main{
struct point y={0,0};
getstruct(&y);
}
struct point getstruct(struct point)
scanf(“%d %d”,&p->x,&p->y);
printf(“%d %d”,p->x,p->y);
return p;\使用结构指针来使得结构变量变化
