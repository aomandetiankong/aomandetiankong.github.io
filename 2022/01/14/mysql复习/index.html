Union联合查询，从表中获取不同的值
SELECT 列名 FROM 表名
UNION
SELECT 列名 FROM 表名
ORDER BY 列名;
UNION 不能用于列出两个表中所有的country。如果一些网站和APP来自同一个国家，每个国家只会列出一次。UNION 只会选取不同的值。请使用 UNION ALL 来选取重复的值！（在union后加all）
下面的 SQL 语句使用 UNION ALL 从 “Websites” 和 “apps” 表中选取所有的中国(CN)的数据（也有重复的值）：
SELECT country, name FROM Websites
WHERE country=’CN’
UNION ALL
SELECT country, app_name FROM apps
WHERE country=’CN’
ORDER BY country;

Mysql分组
SELECT name, COUNT() FROM 表名 GROUP BY 列名;
分为name count（）两组count（）按顺序从1开始依次加1
WITH ROLLUP 可以实现在分组统计数据基础上再进行相同的统计（SUM,AVG,COUNT…）。
例如我们将以上的数据表按名字进行分组，再统计每个人登录的次数：
SELECT 列名, SUM(次数统计基础的列名) as signin_count FROM 表名 GROUP BY name WITH ROLLUP;
其中记录 NULL 表示所有人的登录次数。

表与表的连接查询（取两个表中相同部分）

SELECT a.runoob_id, a.runoob_author, b.runoob_count FROM runoob_tbl a INNER JOIN tcount_tbl b ON a.runoob_author = b.runoob_author;
SELECT a.runoob_id, a.runoob_author, b.runoob_count FROM runoob_tbl a, tcount_tbl b WHERE a.runoob_author = b.runoob_author;
